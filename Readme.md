
# strikethrough

Tiny JavaScript helper and cmd line utility to generate
"strikethrough text" with just character codes. L̶i̶k̶e̶ ̶t̶h̶i̶s̶.

## How to use

```js
var str = require('strikethrough')('some text');
console.log(str);
```

## Installation

To install `st(1)`:

```js
$ npm install -g strikethrough
$ st "some text"
```
